OutStream In-article Player – JS

The Rubicon Project OutStream tag is a JavaScript tag in the format below. The JavaScript tag will load the OutStream video player and render it when it is 50% or more in-view for the user, pause when it’s more than 50% out of view, and close when the ad completed playing.

<script type="text/javascript">
     (function() {
       function loadIt() {
         console.log("DOM fully loaded and parsed");
         // Enter OutStream configuration options HERE //
         var configParams = {
           "width": 640,
           "height": 360,
           "k_pos": "after",
           "k_placement": "a/top",
           "k_align": "center"
         };
         // End OutStream configuration options //
         var doc = top.window.document;
         var as = doc.createElement('script'),
           aT = '//video-ads-apex.rubiconproject.com/apex/1234/5678/90123/203/apex.js?r=' + Math.random() * 10000000000000000;
         as.type = 'text/javascript';
         as.src = aT;
         as.async = true;
         if (as.textContent) {
           as.textContent = JSON.stringify(configParams);
         } else {
           as.innerText = JSON.stringify(configParams);
         }
         doc.body.appendChild(as);
       }
       if (document.readyState === "complete" || (document.readyState !== "loading" && !document.documentElement.doScroll)) {
         loadIt();
       } else {
         document.addEventListener("DOMContentLoaded", loadIt);
       }
     }());
 
</script>

Placement Options

Below are the options that can be used for the placement of the OutStream ad unit within the page. The parameter to specify the placement is called “k_placement”: – followed by the string indicating where the ad unit should render (i.e. within a specific <div> or automatically detecting it within the HTML)

Example:
“k_placement”: “g/#my-div”

Note: The string in the placement must either start with “g/” (for a specific div) or with “a/” for automatically detecting the placement in the HTML on the page.

Specific <div> tag

Option The OutStream video player will render at…
“k_placement”: “g/#OutStream” <div id=OutStream>
This <div> tag must be hard coded onto the page
“k_placement”: “g/.OutStream” <div class=OutStream>
This <div> tag must be hard coded onto the page
“k_placement”: “g/p:nth-of-type(2)” … specified paragraph (paragraph 2 in this example). To specify paragraph 3, use
“k_placement”: “g/p:nth-of-type(3)”
etc.

Automatically Detect Placement

Option The OutStream video player will render at…
“k_placement”: “a/top” … the top of the page content
“k_placement”: “a/middle” … the middle of the page content
“k_placement”: “a/bottom” … the bottom of the page content

Multiple placements

In the event where the same OutStream tag gets trafficked to site(s) with multiple page designs, where a one placement setting does not cover all pages (page designs), multiple placements can be added. Each needs to be separated by a comma:

Example:
“k_placement”: “a/middle,g/p:nth-of-type(2),g/#OutStream”

The OutStream ad unit will render on the placement that is found first on the page.

Passback

Should Rubicon Project not be able to fill the OutStream ad opportunity, the OutStream tag will call the passback function defined by the publisher. This function call will contain the original OutStream container as the argument. Inside this function, the publisher should add whatever code is required to instantiate a secondary player option to fulfill the ad slot opportunity.

There are 2 updates to the ad tag to add a passback:

  • Add “k_passback:” to the OutStream configuration, specifying the name of the passback function (line 24)
  • Add the script code from line ‘” // Begin passback function //’ untile line “// end of passback function” (lines 44-62)

           as.innerText = JSON.stringify(configParams);
         }
         doc.body.appendChild(as);
         // Begin passback function //
         top.window.myPassback = function (passbackContainer) {
           // insert callback function here
           console.log("Call passback", passbackContainer);
           var passbackCode = '<script src="//a.teads.tv/page/12345678/tag" async="true"><' + '/script>';
          
           if (passbackContainer) {
